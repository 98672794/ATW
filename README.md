### 介紹

ATW商店輔助程式是一款協助企業處理日常運營的軟件。其功能包括：

    - whatsapp自動客服機器人
    
    - 自動搵客(搜尋網站 or FACEBOOK)
    
    - 自動下載網頁模板
    
    - 生成短網址(bitly)
    
    - chatGPT
      
    -- 更多功能正在開發中


以每日20元的費用，這個程式將協助您解決重複人手工作的痛點。
